  OUTPUT: Comparison statistics

================================================================================
                       TROUBLESHOOTING
================================================================================

ISSUE: "No checkpoint found"
SOLUTION: Use auto-training
  $ python run_eda.py --num-epochs 5 --num-trajectories 20

ISSUE: "CUDA out of memory"
SOLUTION: Reduce batch size or use CPU
  - Edit src/quick_train.py, line 50
  - Or let it auto-fallback to CPU

ISSUE: "Results directory not found"
SOLUTION: Check permissions
  $ chmod -R 755 results/
  Or specify custom directory:
  $ python run_eda.py --results-dir ./my_results

ISSUE: "Strange trajectories"
SOLUTION: Train longer
  $ python run_eda.py --num-epochs 10 --num-trajectories 20

More help: See EDA_README.md - Troubleshooting section
